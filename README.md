# 🪙 Coin Tossing Game (Python)

## 📌 Overview
This is a simple command-line coin tossing game built with Python.  
It simulates flipping a coin and randomly returns either **heads** or **tails**.

---

## ✨ Features
- ✅ Random coin toss simulation
- ✅ Interactive user input
- ✅ Replay option
- ✅ Beginner-friendly code

---

## 🛠️ Tech Stack
- Python
- Built-in `random` module

---

## ⚙️ How It Works
- The user types **"toss"** to flip the coin
- The program randomly selects:
  - `heads`
  - `tails`
- The result is displayed instantly
- User can choose to play again
